## 闵行区文件格式分析

```

    根据《上海市教育委员会关于2025年本市高中阶段学校招生工作的若干意见 》（沪教委基〔2025〕5号）要求，现
对2025年上海市高中名额分配到区招生计划公示 。

序号

学校
招生代码

学校名称

委属名额
区属名额数
...

062001 062002 ...
上海市市西中学
上海市育才中学
...
```

**特点**：
- 使用 `序号` 列，而非简单的学校名称列表
- 有 `委属名额`（全市）和 `区属名额数`（各校分配给各区的总数）两列
- 结构更复杂，不是标准的「代码 | 名称 | 名额」格式

## 建议

闵行区的格式与其他区不同，需要**专门的解析脚本**。

可以考虑：
1. 人工处理闵行区数据（Excel格式可能更准确）
2. 使用 OCR 直接识别表格结构
3. 从官方网站获取数据


**特点**：
- **两段式结构**：
  1. 第一段：初中学校列表（无代码）
  2. 第二段：高中代码 → 招生学校名称 → 计划数
- **重复问题严重**：markdown中学校名称和计划数重复多次
- **重要发现**：
  - 行84：交大附中附属嘉定德富中学（2024学校）
  - 行878：交大附中附属嘉定洪德中学（2025新增学校）
  - 行470+：上海师范大学附属中学嘉定新城分校

**已验证**：
- ✅ 交大附中附属嘉定洪德中学 - 真实存在（用户确认）
- ✅ 上海师范大学附属中学嘉定新城分校 - 已在2025学校数据中
- ℹ️ 交大附中附属嘉定德富中学 - 2024年学校（JD0015）

**建议**：
1. 使用专门的解析脚本：`extract_2025_quota_to_school_jiading.py`
2. 需要处理学校名称重复和跨行问题
3. 确认"洪德"和"德富"是两所不同学校（非OCR错误）
