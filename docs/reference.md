## 政策规则详解

### 1. 2025年最低投档控制分数线

| 类别 | 最低投档控制分数线 | 说明 |
|------|-------------------|------|
| 普通高中自主招生录取 | 605分 | 含委属市实验性示范性高中、市特色普通高中 |
| 名额分配综合评价录取 | 605分 | **含综合素质评价50分，总分800分** |
| 普通高中统一招生录取 | 513分 | 1-15平行志愿 |
| 中本贯通录取 | 513分 | 须达普高线 |
| 五年一贯制和中高职贯通录取 | 400分 | - |
| 普通中专录取 | 350分 | - |

### 2. 各科目分数构成

| 考试科目       | 总分  | 备注                                                                 |
|----------------|-------|--------------------------------------------------------------------|
| 语文           | 150分 | 闭卷笔试                                                           |
| 数学           | 150分 | 闭卷笔试                                                           |
| 外语           | 150分 | 笔试140分（含听力25分）+ 听说测试10分                              |
| 道德与法治     | 60分  | 统一考试30分（开卷笔试）+ 日常考核30分                             |
| 历史           | 60分  | 统一考试30分（开卷笔试）+ 日常考核30分                             |
| 综合测试       | 150分 | 物理70分 + 化学50分 + 跨学科案例分析15分 + 物理实验操作10分 + 化学实验操作5分 |
| 体育与健身     | 30分  | 统一考试15分 + 日常考核15分                                       |

**学业考试总成绩满分：750分**
**名额分配批次总分：750分 + 50分综合素质评价 = 800分**

### 3. 同分比较规则（6位序）

| 位序 | 比较内容 | 说明 |
|------|----------|------|
| 第1位序 | 同分优待 | 经公示享受同分优待的学生优先 |
| 第2位序 | 综合素质评价成绩 | 50分满分，高者优先 |
| 第3位序 | 语数外三科合计成绩 | 语文+数学+外语，高者优先 |
| 第4位序 | 数学成绩 | 高者优先 |
| 第5位序 | 语文成绩 | 高者优先 |
| 第6位序 | 综合测试成绩 | 含物理、化学、跨学科案例分析、实验操作，高者优先 |

### 4. 名额分配招生计划比例

| 学校类型 | 名额分配到区 | 名额分配到校 | 说明 |
|----------|-------------|-------------|------|
| 委属市实验性示范性高中 | 80% | 20% | 该批次招生计划约占本校总计划的65% |
| 区属市实验性示范性高中 | 30% | 70% | 到区计划中90%-95%分配到外区 |

### 5. 招生录取顺序

```
自主招生录取 → 名额分配综合评价录取 → 统一招生录取
```

### 6. 名额分配到校填报资格

| 学生类型 | 是否可填报 | 条件 |
|---------|-----------|------|
| 不选择生源初中在籍在读满3年的应届初三学生 | ✅ | 须由毕业学校认定，公示5个工作日，报区教育局备案 |
| 选择生源初中（如民办初中）的学生 | ❌ | 不具备填报资格 |
| 往届生 | ❌ | - |
| 返沪生 | ❌ | - |
| 跨区报名的应届初三学生 | ❌ | - |

### 7. 各批次志愿设置

#### 名额分配综合评价志愿

| 类别 | 志愿数 | 填报资格 | 总分 |
|------|--------|----------|------|
| 名额分配到区 | 1个志愿 | 完成中招报名的学生 | 800分 |
| 名额分配到校 | 2个平行志愿 | 仅限不选择生源初中在籍在读满3年的应届生 | 800分 |

#### 统一招生志愿

| 类别 | 志愿数 | 说明 | 总分 |
|------|--------|------|------|
| 1至15志愿 | 15个平行志愿 | 完成中招报名的学生均可填报 | 750分 |
| 区级优秀体育/艺术生 | 1个志愿 | 须通过区级资格确认 | 750分 |
| 征求志愿 | 若干 | 1-15志愿未被录取的学生 | 750分 |

### 8. 平行志愿原则

**分数优先、遵循志愿、一轮投档**

### 11. 志愿填报约束

| 约束类型 | 说明 |
|---------|------|
| 填报时间 | 在学业考试后、成绩发布前进行 |
| 确认方式 | 填报完成后须进行志愿书面确认，签字确认后不可更改 |
| 批次联动 | 若在自主招生录取批次被正式录取，名额分配综合评价志愿和统一招生志愿自然失效 |
| 不可更改 | 市、区招考机构不受理更改志愿、顺延志愿、放弃志愿和补填志愿等申请 |

| 民办高中国际课程班非本市生源 | - |
| 成绩复核 | - |
