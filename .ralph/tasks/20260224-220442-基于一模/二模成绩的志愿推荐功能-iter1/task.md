# 任务：基于一模/二模成绩的志愿推荐功能

## 背景

上海中考模拟系统需要增加一个新功能：根据考生的所在区和一模/二模成绩，智能推荐三个批次（名额到区、名额到校、统一招生）的志愿填报学校。

**一模/二模背景**：
- 一模：初三第一学期末（1月），全区排名定位，总分因区而异（615-700分）
- 二模：4月份进行，最贴近中考难度，总分因区而异（605-615分）
- 核心价值：区域排名和薄弱环节诊断，非绝对分数

**现有数据表**：
- `ref_admission_score_quota_district`：名额分配到区录取最低分数线（总分800分）
- `ref_admission_score_quota_school`：名额分配到校录取最低分数线（总分800分）
- `ref_admission_score_unified`：1-15志愿录取分数线（总分750分）
- `ref_control_score`：最低投档控制分数线

**分数转换**：
- 一模/二模分数需要转换为中考等效分数（750分制或800分制）
- 利用区中考人数和历年数据进行排名估算

## 技术细节

### 分数转换公式

```
中考等效分 = (一模/二模分 / 该区一模/二模总分) × 750
排名估算 = 该区中考人数 × (1 - 考生百分位)
```
