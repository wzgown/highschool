## 设计要求

### 视觉效果

1. **蘑菇云形态**
   - 分数为Y轴，人数为X轴（水平方向展开）
   - 高分段（630-650）形成"蘑菇顶"
   - 中低分段（550-620）形成"蘑菇柄"
   - 整体呈现倒置的蘑菇云形状

2. **动态效果**
   - 粒子系统：每个粒子代表一定数量的考生
   - 粒子从底部（低分）向顶部（高分）缓慢上升
   - 高分区域粒子密集、活跃
   - 低分区域粒子稀疏、缓慢
   - 可考虑"爆炸"效果：从峰值向两侧扩散

3. **配色方案**
   - 主色调：暖色系（橙、红、黄）体现竞争激烈
   - 高分区域：火焰色/金色（成功、希望）
   - 中分段：橙色过渡
   - 低分段：深红/暗色（压力、挑战）
   - 背景渐变：深色背景（深蓝/深紫）突出粒子

4. **交互效果**（可选）
   - 鼠标悬停显示具体分数段和人数
   - 滚动视差效果
   - 响应窗口大小变化

### 技术要求

1. **性能优化**
   - 使用 Canvas 2D 渲染（非 WebGL，兼容性更好）
   - 粒子数量控制在合理范围（建议 500-2000）
   - 使用 requestAnimationFrame
   - 支持 prefers-reduced-motion

2. **组件结构**
   - 创建独立的 Vue 组件：`ScoreDistributionBackground.vue`
   - 组件应接受配置 props（粒子密度、动画速度等）
   - 数据通过 props 传入，便于未来数据更新

3. **样式要求**
   - 组件绝对定位，z-index: 0
   - 内容层 z-index: 2
   - 支持 fullPage 模式（覆盖整个页面）
   - 响应式设计，移动端适配

## 设计理念

这个背景不仅仅是装饰，而是要传达：
- **竞争烈度**：高分段密集的考生分布
- **压力感**：蘑菇云的视觉冲击
- **希望与挑战并存**：暖色调的渐变
- **数据可视化**：真实的成绩分布数据
