## 已整理数据

### 1. 参考人数 (quota/)
- **exam_counts_2025.csv**: 2025年各区中考人数
  - 16个区，共约11.6万人
  - 浦东新区最多（30,447人），崇明区最少（2,590人）

### 2. 名额分配 (quota/)
- **quota_to_district_2024.csv**: 2024年名额分配到区数据
  - 74所市实验性示范性高中
  - 全市分配计划数

- **quota_to_school_2024.csv**: 2024年名额分配到校数据
  - 全市所有初中学校的配额
  - 每所初中对应的高中及名额数

### 3. 录取分数线 (scores/)
- **cutoff_scores_2024.csv**: 2024年1-15志愿录取分数线
  - 16个区，906条记录
  - 包含学校名称、代码、录取分数线
